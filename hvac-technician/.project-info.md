- Docker containerized

## Usage

```bash
# Interactive mode
docker compose exec crewai python /app/hvac-technician/hvac_expert.py

# Single question
docker compose exec crewai python /app/hvac-technician/hvac_expert.py "your question"

# Quick start menu
./hvac-technician/quick-start.sh
```
